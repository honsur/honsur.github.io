Détection des occlusions artérielles (Première Lecture)

Avant d’examiner le matériel pédagogique fourni dans ce module, nous commencerons par une interprétation initiale de 10 angioscans pour fournir une base de référence.

Voici le lien pour les cas et le formulaire

Pour accéder aux cas, cliquez sur le lien hypertexte avant chaque série de questions, comme dans l’image fournie. Les cas sont hébergés sur un système PACS ouvert, accessible uniquement sur l’intranet de l’hôpital pour le moment. Pour vous connecter utilisez le nom d’utilisateur resident4 et le mot de passe Recherche21.

Pour chaque cas, lorsque vous aurez vu les images, il faudra remplir un formulaire automatisé, en indiquant vos trouvailles

Il est possible de faire une pause et de revenir à l’enquête. Assurez-vous simplement d’entrer un e-mail valide pour recevoir le lien vers votre questionnaire partiellement rempli.

Un lien de téléchargement vers vos réponses peut être obtenu sur la dernière page de l’enquête.
