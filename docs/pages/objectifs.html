Objectifs

Objectifs d’apprentissage

Après avoir parcouru ce module, le résident junior de radiologie devrait être en mesure de :

  1. Connaître les indications d’imagerie en contexte d’accident vasculaire cérébral (AVC)
  2. Connaître les principes guidant le choix des modalités d’imagerie en AVC
  3. Interpréter un scan cérébral C- dans le contexte spécifique de l’AVC aigu
  4. Interpréter un angioscan cérébral dans le contexte spécifique de l’AVC aigu
  5. Interpréter un angioscan cervical dans le contexte spécifique de l’AVC aigu
  6. Communiquer rapidement à l’équipe traitante les trouvailles pertinentes à l’imagerie
